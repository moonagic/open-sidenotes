# Changelog

## [0.6.1](https://github.com/mlhiter/open-sidenotes/releases/tag/v0.6.1) (2025-11-25)

### Features
- Task list support with `- [ ]` and `- [x]` syntax
- Markdown editor with real-time rendering
- Settings panel with customization options
- Session persistence for last opened note

- Fixed edit bug
- Fixed link and quote display issues
